Re:filter — это попытка создать актуальный список заблокированных доменов и IP-адресов в РФ, а также популярных и заблокированных для пользователей из России.

Позволяет заменить список Ip-адресов используемого пресета на список всех заблокированных Ip-адресов. 
Для замены списка Ip-адресов посетите страницу параметров компонента, выберите нужный список Ip-адресов, откройте [Дополнительные действия] и выберите [Заменить список Ip-адресов]
