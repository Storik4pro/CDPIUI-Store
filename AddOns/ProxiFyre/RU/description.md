Помощник обертывания трафика для компонентов-прокси.

Позволяет обрабатывать UDP трафик для тех компонентов, которые это поддерживают и настраивать список приложений/исключений для проксирования. Не является обязательным для использования компонента-прокси, но значительно упрощает контроль трафика

#### Обратите внимание
Установка этого дополнения может потребовать принять стороннее лицензионное соглашение в процессе установки
